# fp-mu-plugin

**FrankenPress must-use plugin** — slim WordPress integrations for the
FrankenPress stack: URL fixer, Souin cache invalidator, Prometheus metrics,
WooCommerce stderr logger, and a Memcached object cache drop-in.

This is the WordPress-side companion to
[`fp-runtime`](https://github.com/EightOEight/fp-runtime) (the container
image — Caddy + FrankenPHP + Souin) and
[`fp-charts`](https://github.com/EightOEight/fp-charts) (Helm chart for
Kubernetes deployment).
